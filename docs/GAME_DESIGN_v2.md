# 羊蹄山之魂 - 游戏设计文档 v2

## 核心问题
1. 玩法无交互：点击按钮→显示文字→数字变化，太无聊
2. 无地图系统
3. 无战斗系统
4. 剧情推进方式不清晰

## 目标
设计一个有趣、可玩、有深度的卡牌RPG。

## 二、战斗系统

### 玩家行动
1. **攻击** - 基础伤害 = 力量 × 2 + 武器加成
2. **技能** - 消耗MP，造成范围伤害或特殊效果
3. **防御** - 本回合受到伤害减半，恢复少量HP
4. **道具** - 使用背包中的药水
5. **抽卡** - 触发一次事件牌效果
6. **逃跑** - 一定几率逃跑，失败则承受反击

### 技能示例
| 技能名 | MP消耗 | 效果 |
|---|---|---|
| 重击 | 10 | 造成150%伤害 |
| 火焰术 | 15 | 造成火属性伤害 |
| 治愈术 | 20 | 恢复30% HP |
| 连击 | 5 | 连续攻击2次 |
| 战吼 | 0 | 本场战斗+20%攻击 |

### 怪物属性
- 等级：1-20
- HP：10 × 等级
- 攻击：等级 × 1.5
- 防御：等级 × 0.8
- 掉落：金币、经验、道具

### 伤害公式
`实际伤害 = max(1, (攻击方.攻击 - 防御方.防御/2) × 随机系数(0.8-1.2))`

## 三、卡牌系统

### 卡牌类型
1. **主线剧情卡** - 推进核心故事，解锁新区域
2. **支线任务卡** - 触发支线任务，获得奖励
3. **遭遇卡** - 随机遭遇：商人、旅人、野怪、宝藏
4. **战斗卡** - 强制进入战斗，胜利后获得丰厚奖励
5. **增益卡** - 获得临时属性提升
6. **减益卡** - 面临挑战DEBUFF

### 卡组管理
- 每进入新区域，从牌堆顶抽1张
- 战斗胜利额外抽1张
- 特殊NPC可交换卡牌

### NPC发牌员
| NPC | 位置 | 特色卡组 |
|---|---|---|
| 哨兵队长 | 边境哨站 | 主线剧情卡为主 |
| 冒险者公会接待员 | 王城 | 任务卡+战斗卡 |
| 酒馆老板娘 | 边境小镇 | 遭遇卡+增益卡 |
| 神秘商人 | 随机出现 | 交换卡牌 |

## 四、成长系统

### 玩家属性
- **等级**：1-20，每级所需经验递增
- **HP**：基础100，每级+20
- **MP**：基础50，每级+10
- **力量**：影响物理攻击
- **智力**：影响技能伤害和MP上限
- **敏捷**：影响行动顺序和逃跑率
- **魅力**：影响NPC对话选项和商店价格

### 升级奖励
每升1级：+2点自由属性，可学习新技能

### 技能树
- Lv.1: 重击
- Lv.5: 防御姿态、火焰术
- Lv.10: 连击、治愈术
- Lv.15: 战吼、冰霜术
- Lv.20: 终极技能（各职业不同）

## 五、交互改进

### 问题：当前的"点击→文字→数字"太无聊
### 解决方案：
1. **场景探索**：点击地图上的场景名称进入
2. **对话选择**：NPC对话提供选项，不同选项影响关系和剧情
3. **实时战斗UI**：进入战斗后显示专用战斗界面，动画+音效
4. **卡牌展示**：抽牌时有翻牌动画，牌面显示效果
5. **状态变化反馈**：数字变化时有飘字动画（+10 HP, -5 MP）
