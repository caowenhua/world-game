# 羊蹄山之魂 - 游戏架构文档 v3.0

> "卡牌不是战斗工具，卡牌是世界本身。"

---

## 一、核心问题反思

### 当前版本的问题
1. ❌ 地图是附属品，不是卡牌系统的载体
2. ❌ 发牌员只是一个NPC，不是机制
3. ❌ UI太粗糙，像半成品
4. ❌ 战斗和卡牌脱节
5. ❌ 没有大局观，头痛医头

### 需要深刻理解的本质
**卡牌推进世界 = 每张卡牌都是世界的一个碎片**

- 你抽到一张"古龙的鳞片" → 世界告诉你古龙的传说
- 你击败一只怪物获得"暗影碎片" → 解锁暗影区域的故事
- 你和NPC对话获得"命运之卡" → 命运之轮开始转动

## 二、世界架构

### 世界分层

```
┌─────────────────────────────────────────┐
│           第一章：觉醒之地              │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  │
│  │新手村│→│森林 │→│洞窟 │→│小镇 │  │
│  │Lv1-3│  │Lv3-5│  │Lv5-7│  │Lv7+ │  │
│  └─────┘  └─────┘  └─────┘  └─────┘  │
│     ↓        ↓        ↓        ↓        │
│  [卡牌]  [卡牌]  [卡牌]  [卡牌]      │
│  引导包  森林套  洞窟套  小镇套        │
└─────────────────────────────────────────┘
                  ↓ 集齐关键卡牌
┌─────────────────────────────────────────┐
│          第二章：王国风云（未解锁）       │
│  需要：集齐"觉醒者"卡组 + 等级10         │
└─────────────────────────────────────────┘
```

### 地图设计原则
1. **区域不是地图，是章节** - 每个大区域是一个故事章节
2. **区域有卡牌门槛** - 没有特定卡牌无法进入
3. **探索奖励卡牌** - 每个角落都可能藏着一张卡
4. **地图是动态的** - 你的卡牌收集进度改变地图

## 三、卡牌系统架构

### 3.1 卡牌分类

```
卡牌
├── 角色卡（Character Cards）
│   ├── 主角：浪人剑客
│   ├── NPC：陈岳、苏婉儿、老猎人...
│   └── 怪物：史莱姆、灰狼、哥布林...
│
├── 技能卡（Skill Cards）
│   ├── 攻击类：重击、火焰术、雷电术
│   ├── 防御类：护盾、闪避、反射
│   └── 辅助类：治愈、增益、召唤
│
├── 剧情卡（Story Cards）
│   ├── 主线：千年契约、觉醒者之血
│   ├── 支线：猎人往事、矿工悲歌
│   └── 隐藏：亡眼印记、虚渊低语
│
└── 道具卡（Item Cards）
    ├── 装备：破魔剑、魂晶项链
    ├── 消耗：生命药水、魔法药水
    └── 特殊：影子契约、皇家令牌
```

### 3.2 卡牌稀有度
- ⭐ 普通 - 到处都有
- ⭐⭐ 稀有 - 完成特定任务
- ⭐⭐⭐ 史诗 - 击败精英怪物
- ⭐⭐⭐⭐ 传说 - 完成章节主线
- ⭐⭐⭐⭐⭐ 神话 - 世界级事件

### 3.3 卡牌获取机制

```
世界事件
│
├── 击败怪物 → 卡牌碎片 → 合成卡牌
│   └── 击败灰狼 → "狼牙"碎片 ×3 → "灰狼"角色卡
│
├── 探索地图 → 发现隐藏 → 获得卡牌
│   └── 探索矿洞深处 → 发现"矿工日记" → 剧情卡
│
├── NPC对话 → 选择分支 → 获得卡牌
│   └── 听完老猎人的故事 → 选择"相信他" → "信任"增益卡
│
├── 野外遭遇 → 随机事件 → 卡牌机会
│   └── 遇到流浪商人 → 用金币购买 → 随机卡包
│
└── 打败BOSS → 章节奖励 → 传说卡
    └── 击败"亡灵巫妖" → "觉醒者之心"传说卡
```

## 四、发牌员机制（核心！）

### 4.1 发牌员不是一个人，是无处不在的机制

**发牌员的本质：在正确的时间、正确的地点，给玩家正确的机会。**

```
发牌场景举例：
│
├── 战斗后发牌
│   └── 击败史莱姆 → 10%概率掉落"史莱姆卡牌"
│
├── 探索时发牌
│   └── 走进古老遗迹 → 自动触发"遗迹卡池"抽取
│
├── NPC发牌
│   └── 和老猎人对话 → "你想听我的故事吗？" → 是 → 获得"猎人传说"
│
├── 怪物发牌
│   └── 击败稀有怪"狼王" → 强制发牌 → "狼王卡"或"狼魂"
│
├── 事件发牌
│   └── 天空出现异象 → 所有人都在讨论 → 点击对话 → 获得"永夜预兆"
│
└── 环境发牌
    └── 在特定地点（废墟、神殿）→ 自动获得"环境卡"
```

### 4.2 发牌规则引擎

```typescript
interface CardDealerRule {
  trigger: 'combat_end' | 'explore' | 'npc_talk' | 'monster_kill' | 'event' | 'location';
  condition: (gameState: GameState) => boolean;
  cardPool: string[]; // 可抽取的卡池
  dropRate: number; // 概率 0-1
  storyContext: string; // 这次发牌的剧情背景
  dialogueBefore?: string; // 发牌前对话
  dialogueAfter?: string; // 发牌后对话
}
```

## 五、战斗系统架构

### 5.1 战斗不是孤立的，是卡牌系统的表演

```
战斗流程：
1. 遭遇怪物 → 显示怪物卡牌信息
2. 战斗开始 → 展示双方手牌
3. 回合制 → 玩家出卡牌 → 怪物出卡牌
4. 伤害计算 → 基于卡牌效果
5. 战斗结束 → 根据表现发放奖励卡牌
```

### 5.2 卡牌驱动的战斗

```
玩家手牌（5张）
├── [普通攻击] - 稳定伤害
├── [火焰术] - 需要"魔力"资源
├── [防御姿态] - 减伤
├── [治愈术] - 恢复HP
└── [必杀技] - 高伤害但有条件

怪物也有手牌！
├── [撕咬] - 普通攻击
├── [咆哮] - 恐惧（跳过玩家回合）
└── [狂暴] - 攻击力上升
```
